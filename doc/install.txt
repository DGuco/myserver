1 服务器环境配置
    # 服务器开发环境  : Ununtu
    # 安装clion
    # gcc版本6.2.0 g++版本 6.2.0
    # 服务器依赖三方库：spdlog,protobuf,openssl
    # openssl安装(Ubuntu):
        sudo apt-get install openssl
        sudo apt-get install libssl-dev
    # spdlog protobuf 解压压缩文件后进入目录执行
        ./configure
        make & make install
